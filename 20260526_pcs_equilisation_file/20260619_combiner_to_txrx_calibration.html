combiner_to_txrx_calibration.py

v1 jmitchell modes: TX / RX python 3.12 · numpy · pandas · matplotlib source @ v1
Inputs — 3 files
  • 1 × combiner CSV — headerless (TX: 9-col, freq in MHz · RX: 6-col, freq in GHz)
  • 2 × reference calibration CSVs — Kev's rig, block format, one per beam
1

Load & normalise combiner data

Both formats reduce to one schema: PCS · TLM · beam · freq_GHz · gain. Null markers (−100.1) dropped.

TX: gain split across two beam columns  ·  RX: single value column (SNR)

runs per beam (1, 2)
2

Aggregate repeats

Median per (PCS × frequency) across the repeat axis.

TX: TLM × test  ·  RX: TLM only

3

Baseline B

B = mean of all combiner points for the beam. Computed from combiner data only — the reference rig is never used here.

4

Attenuation

attenuation = median − B per PCS. Positive = above baseline (attenuate down); negative = below.

5

Shift & interpolate

Frequency-shift, then linear interpolation onto the reference rig's grid (np.interp; clips at grid edges).

TX: +3937.5 MHz  ·  RX: +2812.5 MHz

6

Write file + verification plot

Calibration CSV in the reference block format — column C copied verbatim from Kev's file, RX/TX tag and Beam ID set per mode. Four-trace plot for visual sign-off.

Outputs — 4 files
  • 2 × calibration CSVs — same filename as the reference input, written to output_dir, one per beam
  • 2 × verification PNGs — one per beam, timestamped
combiner raw combiner −B shift file (shift + interp) reference rig, unaltered
